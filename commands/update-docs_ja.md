# ドキュメント更新

信頼できるソースからドキュメントを同期します：

1. package.json のスクリプトセクションを読み込む
   - スクリプト参照テーブルを生成
   - コメントから説明を含める

2. .env.example を読み込む
   - すべての環境変数を抽出
   - 目的とフォーマットをドキュメント化

3. docs/CONTRIB.md を生成する：
   - 開発ワークフロー
   - 利用可能なスクリプト
   - 環境セットアップ
   - テスト手順

4. docs/RUNBOOK.md を生成する：
   - デプロイメント手順
   - モニタリングと警告
   - 一般的な問題と修正方法
   - ロールバック手順

5. 古くなったドキュメントを特定する：
   - 90日以上変更されていないドキュメントを検索
   - 手動確認用にリストアップ

6. 差分サマリーを表示
